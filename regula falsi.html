📖 User Guide

1. Langkah Penggunaan

  1. Masukkan fungsi f(x).
  2. Contoh fungsi yang dapat digunakan:
    • x^3-2*x-2
    • cos(x)-x
    • sin(x)-0.5
    • x*cos(x)-1
  3. Masukkan nilai batas bawah (a).
  4. Masukkan nilai batas atas (b).
  5. Pastikan nilai f(a) × f(b) < 0 agar interval mengurung akar.
  6. Masukkan nilai toleransi (ε).
  7. Masukkan jumlah maksimum iterasi.
  8. Klik tombol Hitung Regula Falsi.

2. Membaca Hasil Perhitungan

  • Akar Hampiran menampilkan hasil akar persamaan yang diperoleh.
  • Jumlah Iterasi menunjukkan banyaknya iterasi yang dilakukan hingga memenuhi kriteria berhenti.
  • Tabel Iterasi menampilkan nilai a, b, c, f(a), f(b), f(c), selang baru, lebar selang, dan status setiap iterasi.
  • Kesimpulan memberikan informasi apakah proses iterasi berhasil menemukan akar hampiran.

3. Visualisasi Iterasi

  • Klik tombol Iter 0, Iter 1, Iter 2, ... untuk melihat proses pencarian akar pada setiap iterasi.
  • Informasi nilai a, b, c, f(a), f(b), dan f(c) akan berubah sesuai iterasi yang dipilih.
  • Grafik akan diperbarui sehingga pengguna dapat melihat perubahan posisi garis secant dan titik akar pada setiap iterasi.

4. Toolbar Grafik

  • Zoom digunakan untuk memperbesar tampilan grafik.
  • Pan digunakan untuk menggeser tampilan grafik.
  • Reset Axes digunakan untuk mengembalikan grafik ke tampilan awal.
  • Gambar Camera digunakan untuk download gamabar grafik

5. Catatan

  • Gunakan * untuk operasi perkalian.
  • Gunakan ^ untuk operasi perpangkatan.
  • Pastikan fungsi ditulis dengan format yang benar agar dapat dihitung oleh sistem.
  • Metode Regula Falsi hanya dapat digunakan jika interval awal memenuhi syarat f(a) × f(b) < 0.
